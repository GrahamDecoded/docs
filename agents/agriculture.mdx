# Agricultural Value Chain Agent

The Agricultural Value Chain Agent is a specialized AI agent designed to optimize agricultural operations, crop management, and supply chain coordination. It integrates with various agricultural systems to provide intelligent insights and automation for farming operations.

## Overview

The Agricultural Agent helps farmers, agricultural cooperatives, and agribusinesses optimize their operations through:

- **Crop Planning & Management**: Intelligent crop selection and growth stage tracking
- **Weather Integration**: Real-time weather monitoring and impact assessment
- **Market Intelligence**: Demand forecasting and price optimization
- **Resource Management**: Land allocation and resource optimization
- **Supply Chain Coordination**: Connecting farmers with buyers and logistics providers

## Core Features

### 1. Farmer Profile Management

The agent maintains comprehensive farmer profiles including:

```json
{
  "farmerId": "string",
  "name": "string",
  "location": "string",
  "landSize": "number (acres)",
  "crops": ["array of crop types"],
  "experience": "number (years)",
  "contactInfo": "object",
  "status": "active|inactive"
}
```

**API Endpoints:**
- `POST /api/agents/agriculture/farmers` - Create farmer profile
- `GET /api/agents/agriculture/farmers/{farmerId}` - Get farmer profile
- `PUT /api/agents/agriculture/farmers/{farmerId}` - Update farmer profile
- `DELETE /api/agents/agriculture/farmers/{farmerId}` - Deactivate farmer

### 2. Agricultural Activity Tracking

Track and manage agricultural activities throughout the growing season:

```json
{
  "activityId": "string",
  "farmerId": "string",
  "cropType": "string",
  "landSize": "number (acres)",
  "startDate": "ISO date string",
  "expectedHarvestDate": "ISO date string",
  "stages": ["array of growth stages"],
  "status": "planned|active|completed|cancelled"
}
```

**Growth Stages:**
- **Planting**: Seed selection, soil preparation, planting date
- **Weeding**: Weed management schedule and methods
- **Fertilizing**: Fertilizer application timing and amounts
- **Harvesting**: Harvest planning and execution

### 3. Market Demand Intelligence

Connect farmers with market opportunities:

```json
{
  "demandId": "string",
  "cropType": "string",
  "quantity": "number (tons)",
  "buyerId": "string",
  "buyerName": "string",
  "deliveryDate": "ISO date string",
  "pricePerTon": "number",
  "status": "open|matched|fulfilled|cancelled"
}
```

**Features:**
- Real-time demand monitoring
- Price trend analysis
- Buyer-seller matching
- Contract management

### 4. Land Booking System

Optimize land utilization and crop rotation:

```json
{
  "bookingId": "string",
  "demandId": "string",
  "farmerId": "string",
  "landSize": "number (acres)",
  "cropType": "string",
  "startDate": "ISO date string",
  "endDate": "ISO date string",
  "status": "booked|active|completed"
}
```

## API Usage Examples

### Start Agricultural Agent Session

```bash
curl -X POST "https://agents-service-xxxxx-uc.a.run.app/api/agents/agriculture/session" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "farmerId": "farmer_123",
    "sessionType": "crop_planning",
    "language": "en"
  }'
```

### Get Crop Recommendations

```bash
curl -X POST "https://agents-service-xxxxx-uc.a.run.app/api/agents/agriculture/recommend" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "farmerId": "farmer_123",
    "location": "Nairobi, Kenya",
    "landSize": 5.0,
    "season": "long_rains",
    "soilType": "clay_loam"
  }'
```

### Track Growth Stage

```bash
curl -X POST "https://agents-service-xxxxx-uc.a.run.app/api/agents/agriculture/stages" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "activityId": "activity_456",
    "stage": "weeding",
    "startDate": "2024-03-15",
    "notes": "Manual weeding completed on 2 acres"
  }'
```

## Configuration

### Environment Variables

```bash
# Agent Configuration
AGENT_AGGRICULTURE_ENABLED=true
AGENT_AGGRICULTURE_MODEL=gemini-pro
AGENT_AGGRICULTURE_MAX_TOKENS=4000

# Weather API Integration
WEATHER_API_KEY=your_weather_api_key
WEATHER_API_ENDPOINT=https://api.weatherapi.com/v1

# Market Data Integration
MARKET_API_KEY=your_market_api_key
MARKET_API_ENDPOINT=https://api.marketdata.com/v1

# Database Configuration
FIRESTORE_PROJECT_ID=your_project_id
FIRESTORE_COLLECTION_PREFIX=agriculture_
```

### Agent Settings

```json
{
  "agriculture": {
    "enabled": true,
    "model": "gemini-pro",
    "maxTokens": 4000,
    "temperature": 0.7,
    "language": "en",
    "weatherUpdateInterval": 3600,
    "marketUpdateInterval": 1800,
    "maxRecommendations": 10
  }
}
```
