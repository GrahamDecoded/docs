## üìß **Bulk Email Service**

### **Send Bulk Email Campaign**

Send bulk email campaigns using AWS SES with advanced features like templates, personalization, and analytics.

```bash
curl -X POST "https://your-function-url.com/sendBulkEmail" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "campaign": {
      "templateName": "welcome_series",
      "from": "noreply@yourcompany.com",
      "recipients": [
        {
          "email": "user1@example.com",
          "variables": {
            "firstName": "John",
            "company": "Tech Corp"
          }
        },
        {
          "email": "user2@example.com",
          "variables": {
            "firstName": "Jane",
            "company": "Design Studio"
          }
        }
      ]
    }
  }'
```

**Response:**
```json
{
  "success": true,
  "campaignId": "campaign_1234567890",
  "messageIds": ["msg_001", "msg_002"],
  "message": "Bulk email campaign sent to 2 recipients"
}
```

## üåê **Domain Registration Service**

### **Register New Domain**

Register domains through AWS Route 53 with automatic DNS configuration and SSL certificate provisioning.

```bash
curl -X POST "https://your-function-url.com/registerDomain" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "domainName": "yourcompany.africa",
    "contactInfo": {
      "FirstName": "John",
      "LastName": "Doe",
      "OrganizationName": "Your Company Ltd",
      "AddressLine1": "123 Business Street",
      "City": "Nairobi",
      "State": "Nairobi",
      "CountryCode": "KE",
      "PhoneNumber": "+254700000000",
      "Email": "admin@yourcompany.africa"
    }
  }'
```

**Response:**
```json
{
  "success": true,
  "operationId": "op_1234567890",
  "message": "Domain registration initiated successfully",
  "estimatedCompletion": "24-48 hours"
}
```
