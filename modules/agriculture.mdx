# Agriculture Management API

The Agriculture Management API enables ERP clients to manage farms, crop planning, harvest tracking, weather monitoring, and agricultural analytics. This system integrates with the AAARRR pipeline to optimize agricultural operations and maximize yields.

## Authentication

All endpoints require authentication using Firebase ID tokens in the Authorization header:

```
Authorization: Bearer <firebase_id_token>
```

### Get Farms

Retrieve farms with filtering and pagination.

**Endpoint:** `GET /api/agriculture/farms`

**Query Parameters:**

- `soilType` (optional): Filter by soil type

- `size` (optional): Filter by farm size range
- `limit` (optional): Number of farms to return (default: 50)
- `offset` (optional): Number of farms to skip (default: 0)

**Response:**

{
  "success": true,
